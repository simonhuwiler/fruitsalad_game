# Fruit Salad - a Game for your Roccat Vulcan Keyboard
## Installation
Clone this Repo:  
`git clone git@github.com:simonhuwiler/fruitsalad_game.git`  

Install it  
`npm install`  

## Run
`node src/index.js`

## How the game works
Healthy fruit will appear on your keyboard. But be aware of the tomatoes, they are genetically modified and will eat other fruits. Trample them (by pressing the key) before they spread! But donâ€™t destroy the healthy fruit, they are part of your five a day!
